# 隔空手写画板

一个在桌面版 Chrome 本地运行的手势绘画网页。摄像头画面和手部关键点只在浏览器内处理，不上传或保存。

## 本地运行

```bash
npm install
npm run dev
```

打开终端显示的本地地址，点击“启用摄像头”并允许权限。

## 操作

- 五指张开稳定约 0.6 秒：打开画笔配置。
- 单食指：移动指针，悬停选择颜色、粗细或完成。
- 拇指与食指捏合：落笔并绘制；松开抬笔。
- 清除：单食指悬停“清除”预激活，再捏合确认。
- 鼠标可用于配置、关键点显示和清除的调试。

## 检查命令

```bash
npm run typecheck
npm test
npm run build
```
