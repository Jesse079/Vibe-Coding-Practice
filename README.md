# 隔空手写画板

一个在桌面版 Chrome 本地运行的手势绘画网页。摄像头画面和手部关键点只在当前浏览器内处理，不上传、不保存。

## 本地运行

```bash
npm install
npm run dev
```

打开终端显示的本地地址，点击“启用摄像头”并允许权限。

如果按当前固定端口启动，可以使用：

```bash
npm run dev -- --host 127.0.0.1 --port 4173
```

然后访问：

```text
http://127.0.0.1:4173/
```

## 操作

- 五指张开稳定约 0.6 秒：打开画笔配置。
- 单食指：移动指针，悬停选择颜色、粗细或完成。
- 拇指 + 食指捏合：在配置面板中确认选项；在清除按钮预激活后确认清除。
- 食指尖：作为绘画时的画笔位置。
- 拇指 + 中指捏合：落笔并开始绘制；保持捏合移动即可连续绘制；松开后抬笔。
- 清除：单食指悬停“清除”按钮预激活，再用拇指 + 食指捏合确认。
- 退出：点击“退出”按钮会清空画布、关闭摄像头并回到启用摄像头页面。
- 鼠标可用于配置、显示/隐藏关键点和清除等调试操作。

## 当前手势策略

绘画采用“食指定位 + 拇指/中指控制落笔”的方式。这样可以让食指尖保持伸出，减少拇指 + 食指捏合时对食指关键点的遮挡和抖动。

配置和清除仍保留拇指 + 食指捏合作为确认动作，避免绘画手势和按钮确认互相串台。
